MinecraftRPG Kit is an independently produced research artifact based on locally supplied Minecraft plugin binaries and decompiled output. Plugin names and trademarks belong to their respective owners.

The kit contains factual JVM metadata, original summaries, research notes, and independently designed feature compositions. It deliberately excludes:

- plugin JARs and class files;
- decompiled implementation bodies or substantial excerpts;
- proprietary images, models, configuration packs, and other vendor assets;
- credentials, account data, logs, caches, histories, or private runtime data.

Decompilation can distort control flow, generics, annotations, nullability, synthetic constructs, and line mappings. `VERIFIED_BYTECODE` records describe only the supplied artifact. `DERIVED_SOURCE` conclusions are research inferences, not vendor guarantees. Critical behavior must be confirmed with a minimal compiled consumer and an authorized test server.

No license or interoperability conclusion is implied by this kit. Before public redistribution, commercial use, or bundling with a product, review the licenses and terms for every original plugin and dependency. This notice is not legal advice.

The source inventory records relative paths, sizes, and SHA-256 hashes solely to prove that analysis inputs remained unchanged. It does not include file contents.
